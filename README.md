# SSSP AMC Management System

A comprehensive web application for managing Asset Maintenance Contracts (AMC), CAMC, and Ashwasan services.

## Features
- **Asset Master**: Track assets, contract periods, and financial details.
- **Bulk Operations**: Download templates and import assets via Excel.
- **Reporting**: Export asset reports directly to Excel.
- **Live Search**: Advanced filtering by Asset Number, Engineer, Customer, and more.
- **Vercel Ready**: Optimized for deployment on Vercel with MongoDB Atlas integration.

## Tech Stack
- **Frontend**: React.js, Vite, Axios, Lucide React.
- **Backend**: Node.js, Express, Mongoose.
- **Database**: MongoDB (Atlas).
- **Deployment**: Vercel.

## Setup
1. Clone the repository.
2. Install dependencies for both frontend and backend.
3. Set up environment variables (`MONGODB_URI`, `JWT_SECRET`).
4. Run `npm run dev`.
